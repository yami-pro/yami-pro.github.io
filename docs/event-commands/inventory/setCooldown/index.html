设置冷却时间

设置冷却键的冷却时间，它将会在后台自动运行
通过<设置数值>指令可以获取冷却时间和进度

  • 角色：角色访问器
  • 操作
    • 设置
    • 增加
    • 减少
  • 冷却键
  • 冷却时间
提示

同一类物品可以使用相同的冷却键，比如治疗药剂、治疗药剂Plus、治疗药剂Ultra。
冷却键可在枚举窗口[F7]中自定义。
