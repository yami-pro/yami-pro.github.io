设置声像

  • 类型
    • BGM
    • BGS
    • CV
    • SE
  • 声像：-1 ~ 1
    • 0：左右声道均衡
    • -1：左声道100%，右声道0%
    • 1：左声道0%，右声道100%
  • 过渡方式
  • 持续时间
  • 等待结束：等待过渡结束，继续执行后面的指令
tip

根据距离和方向来设置声像，可以听声辩位。
