区域访问器

参数​

  • 类型
    • 事件触发区域：区域对象的事件被触发时，通过访问<事件触发区域>获取这个区域
    • 指定区域ID：获取场景中预设的区域，启用参数(数据ID：只有切换到打开的场景时，才能从当前场景中获取区域列表)
