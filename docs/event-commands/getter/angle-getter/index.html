角度访问器

参数​

  • 类型
    • 朝向指定位置：计算角色朝向目标位置的角度，启用参数(位置)
    • 绝对角度：指定角度，正右方 = 0度，顺时针旋转
    • 相对角度：角色当前角度的相对角度，-180度 = 相反方向
    • 相对方向角度：角色当前动画方向的相对角度，受动画方向数量影响
    • 随机角度：随机生成0 ~ 360度
