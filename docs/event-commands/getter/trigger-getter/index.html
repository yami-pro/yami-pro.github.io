触发器访问器

参数​

  • 类型
    • 事件触发器：触发器对象的事件被触发时，通过访问<事件触发器>获取这个触发器
    • 最新创建触发器：指向最新创建的触发器对象，在调用<创建触发器>指令时刷新
    • 触发器变量：获取变量中的触发器对象，启用参数(变量)
