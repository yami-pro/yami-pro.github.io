设置指针事件根元素

  • 操作
    • 设置根元素：设置为根元素后，只会触发这个元素和它的后代元素的指针事件，屏蔽其他元素的指针事件
    • 重置：重置为默认的根容器
tip

给窗口元素设置以下事件是很常用的

  • 自动执行：设置自己为根元素，屏蔽窗口以外的元素事件
  • 销毁事件：重置根元素
