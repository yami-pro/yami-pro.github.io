动画 - 音效动画帧

动画帧​

  • 音效：播放的音频文件
  • 音量：音效的最大音量(0 ~ 1)
提示

通过动画音效层播放的音效，是衰减的。
摄像机中心离动画的位置越远，则音量越小。
如果距离太远，则不会进行播放。
