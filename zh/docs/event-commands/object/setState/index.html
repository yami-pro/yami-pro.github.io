设置状态

设置状态的持续时间，新创建的状态有60秒的持续时间。这时候需要读取自定义的持续时间属性，并将它写入。如此，状态倒计时就会自动进行了。

  • 状态：状态访问器
  • 操作
    • 设置时间
    • 增加时间
    • 减少时间：当状态的时间 = 0时，会被移除
