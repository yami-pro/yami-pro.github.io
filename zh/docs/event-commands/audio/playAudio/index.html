播放音频

  • 类型
    • BGM：只能同时播放一个音频
    • BGS：只能同时播放一个音频
    • CV：只能同时播放一个音频
    • SE：可以同时播放多个音频
    • SE - 衰减：可以同时播放多个音频，常用于战斗音效
      • 声音位置：摄像机中心距离声音位置越远，声音衰减得越多，相关设置在<项目设置>中修改
提示

SE - 衰减：
实际音量取决于开始播放时与摄像机中心的距离，之后不再发生变化
当声音位置距离摄像机超过一定距离时，将不会播放音效
