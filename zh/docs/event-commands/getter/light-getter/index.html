光源访问器

参数​

  • 类型
    • 事件触发光源：光源对象的事件被触发时，通过访问<事件触发光源>获取这个光源
    • 最新创建光源：指向最新创建的光源对象，在调用<创建光源>指令时刷新
    • 指定光源ID：获取场景中预设的光源，启用参数(数据ID：只有切换到打开的场景时，才能从当前场景中获取光源列表)
    • 光源变量：获取变量中的光源对象，启用参数(变量)
