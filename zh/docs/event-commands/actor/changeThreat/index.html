增减仇恨值

  • 角色：角色访问器
  • 目标：角色访问器
  • 操作
    • 增加：增加对目标角色的仇恨值，同时自动将目标角色加入到角色的目标池中
    • 减少：减少对目标角色的仇恨值
  • 仇恨值：可通过<获取目标>指令获取<最大仇恨值>角色
提示

可以按伤害比例，增加角色对目标的仇恨值。
