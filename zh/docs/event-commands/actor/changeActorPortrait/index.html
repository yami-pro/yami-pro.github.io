改变角色头像

  • 角色：角色访问器
  • 模式
  • 完全模式：同时改变图像和裁剪区域
  • 图像模式：仅改变图像
  • 裁剪模式：仅改变裁剪区域
提示

可以将不同角色的头像放到一个图像中，通过改变裁剪区域来切换角色。
或者将一个角色的不同表情头像放到一个图像中，通过改变图像来切换角色，通过改变裁剪区域来切换表情。
