改变通行区域

改变指定角色的可通行区域

  • 角色：角色访问器
  • 通行区域
    • 地面：角色可以在地面移动，无法进入水面和墙块
    • 水面：角色可以在水面移动，无法进入地面和墙块
    • 不受限制：角色可以在任意地形中移动
提示

坐船的时候切换成水面，上岸的时候改成陆地，坐飞艇的时候设置为不受限制。
