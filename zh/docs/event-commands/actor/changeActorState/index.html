改变角色状态

  • 角色：角色访问器
  • 操作
    • 添加：创建指定ID的状态，添加到角色状态列表中（如果已经存在相同状态，先移除它）
    • 移除：移除指定ID的状态
    • 移除实例：读取状态访问器，从角色角色状态列表中移除它
