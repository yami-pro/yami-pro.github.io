平移角色

朝指定方向位移一段距离，可用于位移技能、击退效果、吸引效果等。

  • 角色：角色访问器
  • 角度：角色平移方向的角度
  • 距离：移动的距离
  • 过渡方式
  • 持续时间
  • 等待结束：等待角色平移结束，继续执行后面的指令
