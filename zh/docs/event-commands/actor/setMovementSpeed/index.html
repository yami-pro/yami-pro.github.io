设置移动速度

基础速度 * 速度系数 * 速度系数(临时) = 最终移动速度

  • 角色：角色访问器
  • 属性
    • 基础速度：角色每秒移动的图块数量。
    • 速度系数：通常用于写入角色速度系数。
    • 速度系数(临时)：通常用于施放技能时，角色移动速度受到影响的情况，比如攻击时移动速度减半，不会被写入存档。
