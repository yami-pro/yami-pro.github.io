设置体重

  • 角色：角色访问器
  • 体重：0 ~ 8
    • 体重 = 0时，忽略该角色的碰撞
    • 角色A的体重 : 角色B的体重 = 1 : 1时，A推B的移动速度变成50%
    • 角色A的体重 : 角色B的体重 = 1 : 2时，A推B的移动速度为0%，推不动
    • 角色A的体重 : 角色B的体重 = 2 : 1时，A推B的移动速度为100%，完全没有阻力
