过渡

  • 写入变量：计算起始值到结束值中间的插值，写入到指定变量中
  • 起始值：过渡循环第一次执行生成的过渡值
  • 结束值：过渡循环最后一次执行生成的过渡值
  • 过渡方式：影响插值计算的过渡曲线
  • 持续时间：过渡循环持续的时间
提示

通过本指令，可以实现奖励结算时，数字由小到大的增长，进度条的过渡拉伸动画
