设置游戏速度

通过缩放<增量时间>来影响游戏速度，可以做到时间停止。

  • 速度：游戏的运行速度，同时影响事件<等待>指令的计时
  • 过渡方式
  • 持续时间
  • 等待结束：等待过渡结束，继续执行后面的事件指令
