窗口 - 队伍

队伍列表​

可以通过右键菜单插入新的队伍
队伍可以随意调换顺序，可以把常用队伍设为第一个，这会在一些输入框中被默认选择

  • 队伍颜色：点击颜色方块可编辑队伍颜色，在场景中可以预览
  • 队伍关系：选中队伍跟其他队伍之间的关系是，绿色的笑脸 = 友好，红色的生气脸 = 敌对
